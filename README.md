# Linux Watchdog Timer (WDT)
## Introduction

### Usage:

```
sudo ./watchdog
```

### Output:

```
Device:         /dev/watchdog
Identity:       iTCO_wdt [version 0]
Timeout:        30 seconds
Pre-timeout:    0 seconds
Timeleft:       30 seconds
Last boot:      Power-On-Reset
```
